前端代码
```javascript
// 前端
axios.post('/employee/update?id=1001', {
  name: '张三',
  age: 25
})
// 实际请求：POST /employee/update?id=1001
// Body: {"name":"张三","age":25}
```
后端代码
```java
// 后端
@PostMapping("/employee/{action}")
public Result handle(
    @PathVariable String action,      // 路径参数
    @RequestParam Long id,            // 查询参数
    @RequestBody Employee data        // 请求体参数
) {
    // action = "update"
    // id = 1001
    // data = {name:"张三", age:25}
}
```
## 完整对应关系

|文档中的位置|参数类型|后端注解|前端传参位置|
|---|---|---|---|
|**Path** 区域|路径参数|`@PathVariable`|URL 路径中 `/status/{status}`|
|**Query** 区域|查询参数|`@RequestParam`|URL 问号后 `?id=xxx`|
|**Body** 区域|请求体参数|`@RequestBody`|HTTP Body 中|
